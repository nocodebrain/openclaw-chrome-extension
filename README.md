# OpenClaw Browser Relay - Chrome Extension

This extension allows OpenClaw AI to access and control your browser tabs.

## Installation Instructions

### Step 1: Extract the Extension

If you received a zip file, extract it to a folder on your computer.

### Step 2: Enable Developer Mode in Chrome

1. Open Chrome
2. Navigate to: `chrome://extensions/`
3. Toggle **"Developer mode"** (top right corner) to ON

### Step 3: Load the Extension

1. Click **"Load unpacked"** button
2. Select the folder containing this extension
3. The OpenClaw Browser Relay extension should now appear in your extensions list

### Step 4: Verify Installation

1. Click the OpenClaw icon in your browser toolbar (puzzle piece icon → OpenClaw)
2. Click "Test Connection" - should show "✓ Connection OK"
3. Extension is ready to use!

## How to Use

Once installed, simply tell OpenClaw AI to use your browser:

```
"Open Railway dashboard using my browser"
"Navigate to railway.app in Chrome"
"Use profile='chrome' to access my browser"
```

OpenClaw will automatically connect to this extension and control your browser.

## What It Does

- **Allows OpenClaw to:**
  - View current tab URLs
  - Navigate to pages
  - Click buttons
  - Fill forms
  - Take screenshots
  - Read page content

- **Security:**
  - Only works when you explicitly tell OpenClaw to use your browser
  - Extension must be active (installed)
  - You remain in control - can disable anytime

## Troubleshooting

**Extension not showing up?**
- Make sure Developer Mode is enabled
- Refresh the extensions page
- Try reinstalling

**OpenClaw can't connect?**
- Make sure extension is enabled (toggle is ON)
- Click the extension icon to verify it's active
- Restart Chrome if needed

**Want to disable?**
- Go to `chrome://extensions/`
- Toggle the extension OFF
- Or click "Remove" to uninstall

## Privacy

This extension:
- Does NOT collect or store any data
- Does NOT send data anywhere except to your local OpenClaw instance
- Only activates when you explicitly request browser access from OpenClaw
